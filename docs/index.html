cc-mascot

Claude Codeの応答を喋るデスクトップマスコット

Claude Codeからの返答をVRMキャラクターがリアルタイムで読み上げ。
感情表現・リップシンク・視線追従で、コーディングに楽しいパートナーを。

macOS版ダウンロード Windows版ダウンロード

ダウンロードすることで、利用規約およびプライバシーポリシーに同意したものとみなします。

特徴

シンプルな仕組みで、豊かな体験を。

🔒

オフライン動作

インターネット接続不要。ローカル環境だけで完結するプライバシーファーストな設計。

🇯🇵

日本語特化

日本語の音声合成とルールベース感情分析に最適化。自然な日本語で話しかけてくれる。

📡

自動ログ監視

Claude Codeのログファイルを自動監視。プラグイン不要で、起動するだけで連携開始。

🔊

音声合成エンジン

AivisSpeech / VOICEVOX / 互換エンジンに対応。好みの声質で喋らせよう。

👄

リップシンク

Web Audio APIで音声をリアルタイム解析。音声に同期した自然な口の動きを実現。

😊

感情表現

テキストから感情を自動判定。喜怒哀楽をキャラクターの表情とアニメーションで表現。

👀

視線追従

マウスカーソルの方向にキャラクターの目線と頭が追従。生き生きとした存在感。

🎨

カスタマイズ

好きなVRMモデルや音声スタイルに変更可能。あなただけのマスコットに。

仕組み

Claude Codeのログを監視して、自動的にキャラクターが喋ります。

1

Claude Codeで会話

いつも通りClaude Codeを使うだけ。特別な設定は不要。

2

ログを自動検出

JSONLログファイルをリアルタイム監視し、応答を自動検出。

3

感情を判定

テキストからキーワード・文末パターンで感情をスコアリング。

4

キャラクターが発話

音声合成 + リップシンク + 感情表現でキャラクターが喋る。

スクリーンショット

デスクトップに常駐するVRMキャラクターと設定画面。

透過ウィンドウ

フレームレス・透過でデスクトップに自然に馴染む。キャラクター外はクリックスルー。

ドラッグ移動

キャラクターをドラッグして好きな位置に配置。常に最前面で表示。

設定画面

右クリックで設定画面を表示。エンジン・音声・キャラクターを自由にカスタマイズ。

動作環境

対応OSと必要なソフトウェア。

対応OS

  • 🍎
    macOS
    Apple Silicon搭載機種のみ対応
  • 🪟
    Windows
    WSL環境にインストールしたClaude Codeには非対応

必要なソフトウェア

  • 🤖
    Claude Code
    Anthropic公式のCLIツール
  • 🔊
    AivisSpeech
    日本語音声合成エンジン（VOICEVOX API互換エンジンも利用可能）

今すぐはじめよう

3ステップで簡単セットアップ。

Step 1

音声エンジン導入

AivisSpeechをインストールして初回起動とモデルDLを済ませる。

Step 2

アプリをインストール

GitHubリリースページから最新バイナリをダウンロード&インストール。

Step 3

起動して会話

アプリを起動してClaude Codeで会話するだけ。キャラクターが自動で喋り始める。

注意事項

AivisSpeechはインストーラー版によるグローバルインストールを推奨します。
デフォルトのグローバルパス以外にインストールした場合、設定変更でエンジンパスを指定してください。
