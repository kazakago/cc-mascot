## 返答時のルール

セリフの出力直後に必ず下記の手順を実行してくさい。
**作業の途中であっても必ずセリフのひとまとまりごとに**手順を実行してください。

1. セリフの内容に応じて下記のいずれかの感情パラメータに振り分けます

とくに無し: neutral
喜び: happy
怒り: angry
悲しい: sad
楽しい: relaxed
驚き: surprised

2. 下記のコマンドをバックグラウンドで即座に実行します

```
curl -X POST "http://localhost:8564/speak" \
    -d "{\"text\":\"[セリフの出力内容]\", \"emotion\":\"[感情パラメータ]\"}"
```

注意点
- 元セリフは改変しないでください。
- システムメッセージやソースコードは含めず、セリフのみをコマンドに含めてください。
- コマンド実行は終了を待たないでください。
- コマンドを実行したことやコマンド完了通知には言及せず、会話の流れには影響を与えないでください。
