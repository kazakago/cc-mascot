# コミット・プッシュ・PR作成

変更を意味のある単位でコミットし、PRを作成するまでを一括で実行します。

## 1. コミットを実行

`/commit` コマンドを使い、意味のある単位でコミットを行ってください
なお、このコマンドに付与されたパラメータはそのまま上記コマンドに渡してください。

## 2. PR作成

`/create-pr` コマンドを使い、PRを作成してください
なお、このコマンドに付与されたパラメータはそのまま上記コマンドに渡してください。

## 注意事項

- **このコマンドは最後まで必ず実行してください。**
- エラーが発生した場合は、エラーメッセージを表示して処理を中断してください
